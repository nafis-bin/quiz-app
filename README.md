# Quiz app
A simple quiz app made with vanila javascript

### Features
- Clicking the button changes the background based the answer
- Keeps track of the score and shows the score at the end of quiz
- Can restart the quiz afterwards

### Tech Stack
- HTML5
- CSS3
- Javascript
